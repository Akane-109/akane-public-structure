# Memory Threads UI Capsule (v0.1)

This repository contains the foundational structure for the **Memory Threads API UI**, which visualizes and manages conversation threading for Codex-based Capsule memory systems.

## Structure

- `docs/` - UI specifications (Markdown with YAML header)
- `codex/` - API-integrated Codex description snippet
- `design/` - Figma-ready UI component map in YAML
- `api/` - Memory Threads API definition (OpenAPI)

## Purpose

This structure enables intuitive navigation and replay of conversation threads through a semantic, time-aware interface.
